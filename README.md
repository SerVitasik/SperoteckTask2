#### Executable:
```bash
npm run dev
```

#### Stack:
* React
* SCSS
* NodeJS
* Express


#### Installation of requirements:
```bash
npm install
```
You need to change the name from example-config.js to config.js and create mongodb database and you need to edit mongodbUrl and jwtToken.
